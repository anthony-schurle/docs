1) A k-chromatic graph G is called critical if χ(G − v) < k for every vertex v ∈ G. Determine, with explanation, the critical 3-chromatic graphs.
First, we observe that for any vertex $v$ in a $3$-chromatic critical graph $G$, it must be that $\chi(G - v) = 2$. By the definition of a $k$-critical graph, the removal of any vertex must decrease the chromatic number, and since $\chi(G) = 3$, it follows that $\chi(G - v) < 3$. If $\chi(G - v) = 0$, then $V(G) \setminus \{v\} = \emptyset$, implying $G$ is a single vertex, which is $1$-chromatic and contradicts our initial assumption. If $\chi(G - v) = 1$, then $G - v$ contains no edges. If $G$ were $3$-chromatic in this case, $v$ would have to be adjacent to at least two vertices that are themselves adjacent to one another, which is impossible if $G - v$ is edgeless. Thus, $\chi(G - v)$ must be exactly $2$ for all $v \in V$, meaning that $G - v$ is always bipartite and contains no odd cycles. Since $\chi(G) = 3$, $G$ itself is not bipartite and must contain at least one odd cycle. Let $C$ be an odd cycle in $G$. If there exists any vertex $v \in V(G)$ that is not a part of this cycle $C$, then $C$ remains a subgraph of $G - v$. This would imply that $G - v$ contains an odd cycle and is therefore not bipartite, which contradicts our finding that $\chi(G - v) = 2$ for all vertices. Consequently, every vertex $v$ in $G$ must lie on every odd cycle of $G$. If there were any additional edges or vertices outside a single cycle $C_{2k+1}$, removing a vertex not belonging to those extra components would still leave an odd cycle, or the graph would contain a proper $3$-chromatic subgraph, violating the minimality required for $G$ to be a critical graph. Therefore, $G$ must be an odd cycle $C_{2k+1}$ for $k \ge 1$ and $k \in \mathbb{N}$. $\blacksquare$

2) An n×n matrix with entries from {1, . . . , n} is called a Latin square if every element of {1, . . . , n} appears exactly once in each column and exactly once in each row. Recast the problem of constructing Latin squares as a coloring problem.
Define the Latin matrix $A = \{a_{i,j}\}_{i, j = 1}^n$. We construct a graph $G = (V, E)$ with vertex set $V = \{a_{i,j} \mid 1 \le i, j \le n\}$ and edge set $E = \{\{a_{i,j}, a_{x,y}\} \subseteq V \mid (i = x \lor j = y) \land (i,j) \neq (x,y)\}$. To recast the Latin square as a coloring problem, we assign colors from $\{1, \dots, n\}$ to each vertex. If we assign vertex $a_{i,j}$ the color $p$, then for all neighbors $a_{x,y}$ where $x=i$ (same row) or $y=j$ (same column), we must have $\text{color}(a_{x,y}) \in \{1, \dots, n\} \setminus \{p\}$. This local constraint at $a_{i,j}$ directly parallels the Latin square requirement that $p$ cannot appear elsewhere in its row or column, thereby enforcing that every element of $\{1, \dots, n\}$ appears exactly once in each row and column across the entire matrix. $\blacksquare$

3) Without using Proposition 5.3.1, show that χ ′ (G) = k for every k-regular bipartite graph G.
We proceed by induction on $k$. The base case holds trivially for $0$-regular bipartite graphs, as there are no edges to color. Assume the assertion holds for all $(k-1)$-regular bipartite graphs such that they admit a proper edge coloring using the color set $\{1, 2, \dots, k-1\}$. Consider a $k$-regular bipartite graph $G$. By the properties of regular bipartite graphs (corollary 2.1.3), $G$ contains a $1$-factor $P$. We assign the color $k$ to every edge in $P$; this assignment is valid because $P$ is a matching, ensuring no two edges of color $k$ share a vertex. The remaining subgraph $G - P$ is $(k-1)$-regular and remains bipartite, so by the induction hypothesis, it admits a proper edge coloring with colors $\{1, 2, \dots, k-1\}$. Combining the coloring of $G - P$ with the edges in $P$ yields a proper edge coloring of $G$ using $k$ colors, establishing that $\chi'(G) \le k$. Conversely, since any vertex $v \in V$ is incident with $k$ edges that must each receive a distinct color, we have $\chi'(G) \ge k$. Therefore, $\chi'(G) = k$. $\blacksquare$

4) Determine, with explanation, the value of ex(n, K1,r) for all r, n ∈ N.
To show $ex(n, K_{1, r}) = \lfloor \frac{n(r-1)}{2} \rfloor$, we observe that $K_{1, r}$ exists in a graph $G$ if and only if there is some vertex $v \in V(G)$ with $\text{deg}(v) \ge r$. To avoid $K_{1, r}$, every vertex must satisfy $\text{deg}(v) \le r-1$. Case 1: $n$ is even or $r$ is odd. Then $n(r-1)$ is even, allowing for the construction of an $(r-1)$-regular graph $G$ with $|E| = \frac{n(r-1)}{2}$ edges. If $G$ were not edge maximal, we could add an edge $xy$ for $x, y \in V(G)$. However, since $\text{deg}(x) = r-1$ and $\text{deg}(y) = r-1$ in $G$, the new graph $G' = G + xy$ would result in $\text{deg}(x) = r$ and $\text{deg}(y) = r$. This creates a $K_{1, r}$ centered at $x$ (and $y$), a contradiction. Thus, $ex(n, K_{1, r}) = \frac{n(r-1)}{2}$, which equals $\lfloor \frac{n(r-1)}{2} \rfloor$ since $n(r-1)$ is even making $\frac{n(r-1)}{2}$ an integer. Case 2: $n$ is odd and $r$ is even. The sum of degrees must be even, meaning at least one vertex must have a degree less than $r-1$. We consider a graph $G$ where $n-1$ vertices have degree $r-1$ and one vertex has degree $r-2$, giving $|E| = \frac{n(r-1)-1}{2}$ edges. To verify such a graph exists and is well-defined, we construct it in two steps. First, because $n$ is odd and $r-2$ is even, we can form an $(r-2)$-regular graph on all $n$ vertices. Second, we select $n-1$ of those vertices. Since $n-1$ is even, we overlay a perfect matching on these specific $n-1$ vertices. The vertices involved in the matching now have degree $(r-2) + 1 = r-1$, while the single unselected vertex remains at degree $r-2$. Suppose we add an edge $xy$ to $G$. If neither $x$ nor $y$ is the vertex of degree $r-2$, then both reach degree $r$. Without loss of generality, if $x$ is the vertex of degree $r-2$, it reaches degree $r-1$, but $y$ (which previously had degree $r-1$) now reaches degree $r$. In both scenarios, an edge addition creates a $K_{1, r}$ at a vertex of degree $r$, a contradiction. Finally, to show the equivalence of the edge count, since $n$ is odd and $r$ is even, $r-1$ is odd, meaning their product $n(r-1)$ is an odd integer. Let $n(r-1) = 2k + 1$ for some integer $k$. Then the algebraic edge count is: $\frac{n(r-1)-1}{2} = \frac{(2k+1)-1}{2} = k$ and the floor function bound is $\lfloor \frac{n(r-1)}{2} \rfloor = \lfloor \frac{2k+1}{2} \rfloor = \lfloor k + 0.5 \rfloor = k$. Therefore, $\frac{n(r-1)-1}{2} = \lfloor \frac{n(r-1)}{2} \rfloor$. Thus, $ex(n, K_{1, r}) = \lfloor \frac{n(r-1)}{2} \rfloor$ holds for all $n$ and $r$. $\blacksquare$

5) Show that $t_{r-1}(n) \le 0.5n^2\frac{r-2}{r-1}$ with equality whenever $r - 1$ divides n.
Case 1: $r-1$ divides $n$. We partition the $n$ vertices into $r-1$ equal sets, each containing $\frac{n}{r-1}$ vertices. To find the total degree, we observe that an arbitrary vertex in one partition is adjacent to all vertices in the other $r-2$ partitions. Each such vertex therefore has a degree of $(r-2) \frac{n}{r-1}$. Since there are $n$ total vertices, the sum of all degrees is $n \cdot \frac{n(r-2)}{r-1} = n^2 \frac{r-2}{r-1}$. By the Handshaking Lemma, the edge count is half the total degree, yielding $t_{r-1}(n) = \frac{1}{2} n^2 \frac{r-2}{r-1}$. Case 2: $r-1$ does not divide $n$. The partitions cannot be equal in size; to maximize edges, they must be as balanced as possible (differing by at most one). To justify that balanced partitions strictly maximize the edge count, consider any two partitions in the complete multipartite graph with sizes $a$ and $b$, where $a > b + 1$. The number of cross-edges between these two specific partitions is $ab$. If we move a vertex from the larger partition to the smaller one, the new partition sizes become $a-1$ and $b+1$. The new number of cross-edges between them is $(a-1)(b+1) = ab + a - b - 1$. Since we assumed $a > b + 1$, it follows that $a - b - 1 \ge 1 > 0$. Therefore, $(a-1)(b+1) > ab$. This proves that making the partitions more equal strictly increases the number of edges. Because any unequal partition sizes result in fewer edges than a perfectly balanced configuration, the maximum edge count for any integer partition is strictly bounded above by $\frac{1}{2} n^2 \frac{r-2}{r-1}$. Thus, $t_{r-1}(n) \le 0.5n^2\frac{r-2}{r-1}$ for all $n$. $\blacksquare$
