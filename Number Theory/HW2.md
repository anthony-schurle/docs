($GCD(300, 111)$)
300 = 2(111) + 78
111 = 1(78) + 33
78 = 2(33) + 12
33 = 2(12) + 9
12 = 1(9) + 3
9 = 3(3) + 0
and so we get 3.
($GCD(289, 323)$)
289 = 0(323) + 289
323 = 1(289) + 34
289 = 8(34) + 17
34 = 2(17) + 0
and so we get 17.
($GCD(3939, 10403)$)
3939 = 0(10403) + 3939
10403 = 2(3939) + 2525
3939 = 1(2525) + 1414
2525 = 1(1414) + 1111
1414 = 1(1111) + 303
1111 = 3(303) + 202
303 = 1(202) + 101
202 = 2(101) = 0
and so we get 101.

We have $y = \frac{100 - 17x}{19}$ and one obvious solution is $(x, y) = (-50, 50)$. Everytime x changes by 19, we have a new solution. But y changes by 17 in the other direction. So we get $x = -50 + 19n$ and $y = 50 - 17n$ for every integer n.
